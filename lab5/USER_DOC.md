# Пользовательская документация по программе отсечения отрезков и многоугольников## 1. ВведениеДанное приложение предназначено для визуальной демонстрации алгоритмов отсечения отрезков и многоугольников. Программа позволяет загружать координаты отрезков и отсекающего окна из файла, визуализировать их на координатной сетке и применять различные алгоритмы отсечения.## 2. Системные требования### Минимальные требования:- Qt Framework 5.12 или выше- Операционная система: Windows 7/10/11, Linux, macOS- Оперативная память: 2 ГБ- Процессор: 1.6 ГГц или выше- Разрешение экрана: 1024x768 или выше## 3. Установка и запуск1. Убедитесь, что на вашем компьютере установлен Qt Framework2. Скопируйте файлы проекта в рабочую директорию3. Откройте проект в Qt Creator4. Скомпилируйте и запустите приложение## 4. Формат входных данныхПрограмма принимает текстовый файл следующего формата:```nx1_1 y1_1 x2_1 y2_1x1_2 y1_2 x2_2 y2_2...x1_n y1_n x2_n y2_nxmin ymin xmax ymax```где:- n - количество отрезков- x1_i y1_i x2_i y2_i - координаты i-го отрезка- xmin ymin xmax ymax - координаты отсекающего окна## 5. Использование программы### 5.1 Основной интерфейсИнтерфейс программы состоит из:- Области рисования с координатной сеткой- Отсекающего окна (красный цвет)- Исходных отрезков или многоугольника (синий цвет)- Отсеченных частей (зеленый цвет)### 5.2 Визуализация- Координатная сетка автоматически масштабируется под размер окна- Размер ячейки сетки составляет 20 пикселей- Отсекающее окно отображается красным цветом- Исходные фигуры отображаются синим цветом- Видимые части после отсечения отображаются зеленым цветом### 5.3 Управление масштабомМасштаб отображения контролируется автоматически для оптимального отображения всех элементов.## 6. Возможные проблемы и их решение### Проблема: Файл не открываетсяРешение: - Проверьте права доступа к файлу- Убедитесь в корректности формата файла- Проверьте кодировку файла (должна быть UTF-8)
